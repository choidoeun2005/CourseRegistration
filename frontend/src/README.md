# Sources


| 파일           | 역할                          |
| ---------------- |-----------------------------|
| `main.jsx`       | React 앱을 실행하는 시작 파일         |
| `App.jsx`        | 전체 페이지 구조와 라우팅을 관리          |


| 디렉토리         | 역할                          |
|--------------|-----------------------------|
| `pages/`     | 하나의 화면 단위 페이지 저장            |
| `components/` | 여러 페이지에서 재사용되는 UI 컴포넌트 저장   |
| `hooks/`     | React 커스텀 훅 저장              |
| `api/`       | 백엔드 API 요청 함수 저장            |
| `data/`      | 백엔드 연결 전 사용할 임시 데이터 저장      |
| `styles/`    | CSS 파일 저장                   |
| `utils/`     | 시간 계산, 데이터 변환 등 공통 함수 저장    |
