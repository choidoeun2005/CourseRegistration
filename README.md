# 전체 개발 규칙

## 프론트엔드

* 화면 단위 파일은 `pages/`에 작성합니다.
* 재사용되는 UI는 `components/`에 작성합니다.
* 백엔드 요청 함수는 `api/`에 작성합니다.
* 임시 데이터는 `data/`에 작성합니다.
* 공통 함수는 `utils/`에 작성합니다.

## 백엔드

* API 주소는 `routes/`에 작성합니다.
* 요청과 응답 처리는 `controllers/`에 작성합니다.
* 실제 기능 로직은 `services/`에 작성합니다.
* 데이터 구조는 `models/`에 작성합니다.
* 임시 데이터는 `data/`에 작성합니다.
