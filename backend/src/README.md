# Backend

`backend` 폴더는 서버와 데이터 처리를 담당합니다.
프론트엔드에서 요청한 강의 목록, 사용자 정보, 시간표 데이터 등을 처리합니다.

## 폴더 구조

```txt
backend/
├── package.json
└── src/
    ├── app.js
    ├── server.js
    ├── routes/
    ├── controllers/
    ├── services/
    ├── models/
    ├── db/
    └── data/
```

## 폴더 및 파일 역할

| 파일        | 역할                            |
| -------------- | ----------------------------- |
| `app.js`       | Express 앱의 기본 설정을 담당          |
| `server.js`    | 서버를 실제로 실행                    |

| 디렉토리           | 역할                            |
|----------------| ----------------------------- |
| `routes/`      | API 주소를 정의                    |
| `controllers/` | 요청을 받고 응답을 반환                 |
| `services/`    | 검색, 필터링, 시간표 충돌 확인 등 핵심 로직 처리 |
| `models/`      | 강의, 사용자 등 데이터 구조 정의           |
| `db/`          | 데이터베이스 연결 및 초기 설정 관리          |
| `data/`        | DB 연결 전 사용할 임시 JSON 데이터 저장    |
